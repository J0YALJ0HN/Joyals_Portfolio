Portfolio Projects:

E-commerce "Amazon-like" E-Commerce Website - ClotheLine

github.com/J0YALJ0HN/ClotheLine

ClotheLine is an e-commerce website developed collaboratively by four computer science university students. The platform provides a user-friendly interface for customers to browse and purchase a wide range of clothing items online. It reflects the combined technical expertise and creative skills of the student team, offering a seamless shopping experience

Sample Shop Mobile Application

github.com/J0YALJ0HN/Sample_Shop_App

The application is built in Java and uses Android Studio for development. It interacts with an SQLite database through DBHelper.java to handle user registration, product information, and basket management. The App enables user registration and login functionalities and allows users to view categories and add products to their basket. Admin capabilities include CRUD operations for categories and products.

Blackjack Game Shell Script

github.com/J0YALJ0HN/Blackjack-Game

A Bash script that simulates a blackjack game between a player and a dealer. The script manages bets, deals cards from a shuffled deck, and allows players to hit or stand based on their hand value. It evaluates hands, announces winners, handles blackjack scenarios, and offers users the choice to continue playing or quit after each round, providing an interactive blackjack experience in the terminal.
